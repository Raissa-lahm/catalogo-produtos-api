#  Catálogo de Produtos API

API REST desenvolvida com Node.js, Express e MongoDB para gerenciamento de produtos com autenticação JWT.

##  Tecnologias
- Node.js + Express
- MongoDB + Mongoose
- JWT (autenticação)
- BCrypt (criptografia de senhas)

##  Como rodar o projeto

### 1. Clone o repositório
```bash
git clone https://github.com/Raissa-lahm/catalogo-produtos-api.git
cd catalogo-produtos-api
```

### 2. Instale as dependências
```bash
npm install
```
